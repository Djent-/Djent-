Terminology

LDAP Lightweight Directory Access Protocol
RFC4510 Defines LDAPv3
X.500 Directory Services standard
DIT Directory Information Tree
entry Unit of data in the DIT. Consists of a set of attributes
attribute Attributes are defined in at least one objectClass. Has a type and one or more values.
type Name or description of an attribute.
schema Defines attributes and objectClasses
dn Distinguished Name - unique identifier of an entry. Consists of a Relative Distinguished Name and the parent entry's DN.
object, container, node All mean entry.
cn Common Name
ou Organizational Unit
dc Domain Component
LDIF LDAP Data Interchange Format
RFC2849 Defines LDIF
RTC Real Time Configuration
nis Network Information Services
olc On-line Configuration
mdb Memory-mapped Database
corba Common Object Request Broker Architecture schema
{X} Index of objects with same parent dn created by slapd
Syncrepl Engine which allows LDAP replication to add redundancy to LDAP
CSN Change Sequence Number

Syncrepl Consumer Configuration Terminology

provider Provider server's hostname or IP address
binddn The admin dn
credentials The admin dn password
searchbase The database suffix
olcUpdateRef Provider server's hostname or IP address
rid Replica ID is a unique 3 digit number which dientifies a consumer. A consumer must have at least one rid.

Terminology

access control Management of read/write permissions assigned to users
ACL Access Control List
peercred Something about unix domain socket file descriptors
SASL Simple Authentication and Security Layer
certtool Package used to create a private key for a certificate authority
